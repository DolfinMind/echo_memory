DolfinMind · Echo Memory

Privacy Policy

Effective date: July 18, 2026

Echo Memory is an offline game. It does not collect, transmit, sell, or share personal data.

App and developer

This policy applies to Echo Memory, an Android game published by DolfinMind, including application package com.dolfinmind.echomemory.

Data collection and sharing

Echo Memory does not collect, transmit, sell, or share personal or sensitive user data. It contains no advertising, analytics, tracking, account, social, or backend SDKs. The production Android app requests no internet access or sensitive device permissions.

Data stored on your device

Scores, streaks, play statistics, tutorial completion, daily-challenge status, and game preferences are stored locally only to provide app functionality. This data is never sent to DolfinMind or a third party.

Retention and deletion

Local game data remains until you clear Echo Memory's app storage or uninstall the app. Either action deletes that data. Echo Memory has no user accounts, and DolfinMind cannot retrieve or remotely delete data it never receives.

Security

Echo Memory minimizes privacy risk by operating without a network connection and without collecting personal or sensitive information. Local data is protected by your Android device's security controls.

Children's privacy

Echo Memory is not specifically directed to children and does not knowingly collect personal data from anyone, including children.

Changes to this policy

If a future release adds online services, advertising, analytics, or different data practices, DolfinMind will update this policy and the Google Play Data safety declaration before that release.

Contact

For questions about Echo Memory or this policy, contact DolfinMind at dolfinmind@gmail.com.
